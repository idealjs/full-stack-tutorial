# 使用vite创建react ts工程
React 应用程序是由组件组成的。一个组件是 UI（用户界面）的一部分，它拥有自己的逻辑和外观。组件可以小到一个按钮，也可以大到整个页面。

## 项目初始化
### 1. 创建react+ts项目
按下快捷键`Ctrl + ~`,打开终端，运行以下命令
```
npm create vite@latest
```

根据提示，输入项目名字并选择react，再选择TypeScript

### 2. 打开项目
运行以下命令，进入项目
```
cd react-ts-test1
```

### 3. 安装依赖
运行以下命令，安装依赖
```
npm install
```

### 4. 运行项目
输入以下命令,运行项目
```
npm run dev
```
