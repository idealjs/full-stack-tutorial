# Alias命令的使用
在Linux中，`alias`是一个用于创建命令别名的内置命令；通过`alias`，可以把常用的命令或命令组合设置为简短的别名，从而简化操作。

## 使用方法
### 1. 创建别名
 运行以下命令：            
```
alias 别名='命令'
```
例如：
```
alias ll='ls -l'
```
在命令行中输入`ll`就相当于输入`ls-l`

### 2. 永久别名
在`~/.bashrc`或者`~/.zshrc`文件中配置以下内容：          
```
alias ll='ls -l'
```

### 3. 查看别名
在命令行中输入以下命令：
```
alias
```

### 4. 删除别名
在命令行中输入以下命令：
```
unalias 别名
```
例如：
```
unalias ll
```
这样ll别名就会被删除
